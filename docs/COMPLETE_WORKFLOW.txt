================================================================================
TROUBLESHOOTING FOR USERS
================================================================================

PROBLEM: "ColdEmailEngine.exe is not recognized"
SOLUTION: 
   1. Make sure exe is in the right folder
   2. Open Command Prompt in that folder
   3. Then run: ColdEmailEngine.exe leads.csv


PROBLEM: "File not found" error
SOLUTION:
   1. Check CSV is in data/raw/ folder
   2. Verify filename spelling
   3. Example correct: data/raw/leads.csv
   4. Wrong: leads.csv (from home folder)


PROBLEM: "No emails generated" or mostly blank emails
SOLUTION:
   This is normal if:
   ‚Ä¢ Companies don't follow standard email patterns
   ‚Ä¢ Website data couldn't be scraped
   ‚Ä¢ Some companies use custom domains
   
   The tool tries 8 different patterns. Some always fail.


PROBLEM: "Very slow" or "hanging"
SOLUTION:
   The scraper intentionally goes slow to avoid blocking.
   This is expected and normal.
   
   To speed up: Use --skip-ai option


PROBLEM: "API key error"
SOLUTION:
   Only needed if you want AI personalization.
   Use --skip-ai to skip this feature.


PROBLEM: Application closes immediately
SOLUTION:
   1. Open Command Prompt manually
   2. Navigate to the folder
   3. Type: ColdEmailEngine.exe leads.csv
   4. See actual error messages
