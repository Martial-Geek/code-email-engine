Welcome! This guide will help you use the Cold Email Outreach Engine to

================================================================================
BEFORE YOU START
================================================================================

1. Place your CSV file with leads in the 'data/raw/' folder
   (Your CSV should have columns: first_name, last_name, company, email, website, etc.)

2. Make sure you have your API keys configured for AI features
   (Required only if you want AI personalization)

EXAMPLE WORKFLOW
================================================================================

1. Get your leads as a CSV file (from LinkedIn, Hunter, Apollo, etc.)

================================================================================
COMMON ISSUES
================================================================================

Q: "File not found" error
A: Make sure your CSV is in data/raw/ folder and you typed the filename correctly

Q: Process is slow
A: The scraper is intentionally slow to avoid getting blocked. This is normal.
   Use --skip-ai to speed things up if you don't need AI features.

Q: API key errors
A: Make sure your Google API key is set up in config/settings.py
   This is only needed if you use AI personalization.

Q: Empty email addresses
A: This happens when the email guesser can't find patterns. It's normal for
   some companies. The tool will try multiple patterns.

TIPS FOR BEST RESULTS
================================================================================

1. Clean your data first - remove duplicates, invalid emails
2. Include company website URLs - this is crucial for scraping
3. Use --skip-ai first to understand the process, then run full pipeline
4. Review email templates before sending - customize them for your campaigns
5. Test with 10-20 leads first, then scale to larger lists
